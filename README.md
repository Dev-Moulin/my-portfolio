## Installation and Development

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn

### Installation
```bash
git clone https://github.com/Dev-Moulin/my-portfolio.git
cd my-portfolio
npm install
```

### Development
```bash
npm start
```
The site will be accessible at http://localhost:3000

### Production Build
```bash
npm run build
```

### Deployment
```bash
npm run deploy
```
